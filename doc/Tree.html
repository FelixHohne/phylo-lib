Module Tree

module Tree: sig .. end

The abstract data type of values representing phylogenetic trees. Representation invariant: the phylogenetic tree never contains two species with the same name. In any given tree, all clade_ids must be unique.

type t 
type clade_id = int

The type of clade node identifiers.

exception UnknownClade of clade_id

The clade_id clade_id could not be found in this phylogenetic tree

val empty : t

The empty phylogenetic tree.

val is_empty : t -> bool

is_empty t is true iff. the tree is empty.

val leaf : string -> t

leaf species is the tree consisting of only species.

val size : t -> int

size t is the size of tree t including both clades and species.

val zip : t list -> t

zip trees is the tree with a clade as the root and trees as the children.

val is_equal : t -> t -> bool

is_equal a b is true if a and b are structurally similar, ignoring the order of the children.

val print_tree : t -> unit

print_tree t prints an ASCII-art t to console semi-prettily.
